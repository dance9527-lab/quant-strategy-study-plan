# Review Agent Round 3 回应

## OOT步数

### 对DeepSeek论证的回应

DeepSeek的论证存在概念混淆—— 验证期>训练期只对固定时点训练成立，而因子研究的标准实践是滚动窗口（rolling window），每个OOT步都有其对应的最近5年训练窗口。

重新审视自己坚持20+步的核心动机：追求的是更多市场环境覆盖，而非验证期长度。滚动窗口相邻两期训练集重叠约83%，12步OOT的独立样本数实际有限，20步提供更多市场环境覆盖。

### 三个折中方案评估

- 方案A（5年+12步+分年度）：分年度分析是事后切片，不能替代滚动窗口OOT。⚠️ 有保留
- 方案B（4年+16步）：训练窗口压缩损失有效历史信号。❌ 不推荐
- 方案C（滚动训练+12步）：滚动训练已内置于框架中，实质同义反复。⚠️

### 最终立场/让步

**让步**：放弃必须24步的机械坚持
**坚持**：至少20步（核心理由：更多市场环境覆盖）
**具体立场**：5年滚动训练窗口 + 20步OOT + 分年度IC报告（补充信息）

## 探索性跟踪通道

### 对DeepSeek约束方案的回应

三约束方案（方向一致性60%+不进组合+5%上限）部分但未完全解决问题。

**解决了**：弱信号不直接侵蚀组合；潜在损害规模有限
**未解决**：路径依赖（研究人员倾向于把观察池因子推进正式组合）；60%阈值过低；观察池对PM是否保密

### 替代方案

1. **观察池匿名化**：不公开因子名称，只记录当前有N个因子在研究
2. **方向一致性门槛提升至70%**（McLean & Pontiff基准）
3. **强制冷却期≥6个月**：进入观察池后需2个完整滚动周期才能申请正式评估
4. **重命名为边缘信号日志**：明确这是日志非通道，不进入任何决策流程

### 最终立场/让步

**让步**：接受有约束的观察池机制（而非完全拒绝）
**坚持**：匿名化+门槛≥70%+冷却期≥6个月+重命名+对PM保密

## t-stat 1.5 vs 1.65

### 最终立场

**接受折中：t-stat ≥ 1.645（单尾10%）**

t=1.5对应单尾p≈0.067，t=1.65对应单尾p≈0.05。1.645是单尾10%的标准临界值，更符合因子研究拒绝域不宜过严的实践逻辑——错过真实弱alpha的代价通常低于接受伪alpha。

## 新的让步清单

1. OOT步数：从必须24步让步至至少20步
2. 探索性跟踪：接受有约束的观察池机制（不再完全拒绝）
3. 探索性跟踪：接受进入观察池但不进组合（要求匿名化+冷却期）
4. t-stat：从≥1.5让步至≥1.645

## 新的坚持清单

1. 训练窗口：5年（底线，不接受压缩至4年）
2. OOT步数下限：20步（覆盖更多市场环境）
3. 探索性因子命名：重命名为边缘信号日志，明确非决策性质
4. 观察池：方向一致性门槛≥70%（不接受60%）
5. 观察池：对PM保密
6. 探索性因子冷却期：≥6个月
