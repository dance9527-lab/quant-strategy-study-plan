## DeepSeek 的讨论回应

### 1. 成本核实
- DeepSeek 原报告：0.262%
- Main 质疑：仓库数据显示 0.206%
- DeepSeek 回应：逐项推导确认 0.206%，原报告包含规费万0.5（非仓库数据）
- **共识**：0.206%（基于仓库 equity_cost_history.csv）

### 2. IC门槛推导
- DeepSeek 给出完整公式：IC > 年化成本 / (sigma_returns x sqrt(252))
- 区分"单因子入选门槛"（0.05，配合 t-stat >= 1.65）和"成本覆盖门槛"
- 风控 IC 0.02 标注为经验规则，非严格推导
- **共识**：IC 0.05 为实盘门槛，0.02 为风控经验门槛

### 3. 因子库缺口
- DeepSeek 原报告：换手率变化率和高阶矩因子是缺口
- Main 质疑：如果没有这些因子，S1-M 是否仍可通过？
- DeepSeek 回应：换手率变化率可由已有 turnover_rate 差分构造；高阶矩与波动率共线
- **共识**：缺口因子非必要，S1-M 仍可通过

### 4. 退市股票审计
- DeepSeek 原报告：需要补充退市股票审计
- Main 质疑：如果 equity_master 已包含退市股票，是否还需要？
- DeepSeek 回应：equity_master 包含 319 只退市股票（5.5%），幸存者偏差风险可控
- **共识**：作为 P1 抽查项（10只样本验证），不阻塞 S1-M
